Algunos Trabajos

Sistema de Vídeo Clases (Webinar)

Sistema que permite la transmisión de clases en vivo , haciendo uso de Youtube Live, entre otros servicios, cuenta con un sistema de chat, encuestas, archivos, enlaces, entre otras características, desarrollado haciendo uso de tecnoligías web, tal es el caso de WebSockets.

Vídeo Chat
WebRTC

Sistema de mensajería instántanea haciendo uso de WebSockets (Socket.io) y WebRTC para el manejo de vídeo llamadas, tanto en dispositivos de escritorio como en dispositivos móviles

Sistema de Condicionales

Desarrollo que busca establecer condicionales sobre la visualziación de contenidos, en base a un entorno gráfico que muestre el flujo en que el un usuario desarrollará una actividad

Chat Social
(Android)

Chat Social para dispositivos móviles que hace uso del tipo de aplicación híbrida, en este caso para dispositivos basados en Android , la aplicación además cuenta con un área de minijuegos que busca que el usaurio no sólo se counique sino que también se divierta :)

The Brick Web
Juego

Inspirado en el popuplar juego The Brick Game, The Brick Web trae al mundo web algunos de sus juegos más icónicos de este dispositivo, el cual bajo su eslogan 999 juegos en 1, nos estraga horas de entrenimiento.
Este proyecto pretende crear un espacio para el fácil desarrollo de estos juegos en está ocasión haciendo uso de tecnologías web como es el caso de Canvas

The Brick Web
ElectronJS

Haciendo uso de ElectronJS, se ha creado una versión nativa para equipos con sistema operativo MAC del juego The Brick Web

Take a Picture
getUserMedia

Toma fotos desde tu navegador , "camara" creada en css, inspirada en el modelo polaroid, captura de foto a través de getUserMedia y almacenamiento de la foto haciendo uso de Canvas.

Dots And Boxes
Game

Juego en el cual el usuario jugará versus el PC , con el objetivo de lograr completar la mayor cantidad de cuadrados.

Shoot The Boxes
Google Cardboard

Juego en el que el usuario deberá disparar a una serie de cajas en un mundo virtual, haciendo uso de las Google CardBoard, desarrollado utilizando WebGl/Three.js.

Dots JS

Este juego se hizo con la intención de afianzar los conocimientos del nuevo estándar de Javascript conocido como ES6, puede ser jugado en línea y offline (Service Worker). El repositorio del proyecto se entrega mayor información de las herramientas utilizadas en la elaboración de este juego

Shoot

Inspirado en el juego Neutrino, este juego tiene como objetivo enseñar Canvas , API proporcionada por la actual versión de HTML5, por ser nativa es posible su funcionamiento en cualquier navegador, tanto móvil como de escritorio.

Shoot ha sido desarrollado haciendo uso de la última versión de Javascript conocida como ES6/ES2015, aplicando algunas de sus funcionalidades
